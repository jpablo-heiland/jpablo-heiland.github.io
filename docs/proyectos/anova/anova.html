1 Introducción

Este documento presenta un flujo de trabajo completo para el análisis de un experimento agrícola mediante el Análisis de la Varianza (ANOVA) con R. El conjunto de datos empleado es npk, incluido en la base de R, que registra la respuesta en rendimiento de Pisum sativum a la aplicación de tres nutrientes: Nitrógeno (N), Fósforo (P) y Potasio (K).

El experimento implementa un Diseño en Bloques Completos Aleatorizados (DBCA) con estructura factorial: tres factores, cada uno con dos niveles (presencia vs. ausencia de aplicación). El ensayo comprende 24 unidades experimentales organizadas en 6 bloques de 4 parcelas cada uno. La variable respuesta es el rendimiento, expresado en libras por parcela.

El flujo de trabajo consta de las siguientes etapas:

  1. Carga y exploración inicial del conjunto de datos.
  2. Estadísticas descriptivas por factor.
  3. Especificación y ajuste del modelo ANOVA.
  4. Validación de supuestos: normalidad y homocedasticidad.
  5. Comparación de medias mediante el Test LSD de Fisher.

El resumen permite identificar la estructura del dataset:

  • yield: variable respuesta. El rendimiento varió entre 44.20 y 69.50 lb/parcela. La media (54.88) y la mediana (55.65) son muy cercanas, lo que sugiere una distribución aproximadamente simétrica.
  • block: factor de bloqueo con 6 niveles. La notación 1:4 indica que cada bloque contiene exactamente 4 unidades experimentales.
  • N, P, K: factores independientes con 2 niveles cada uno (0 = sin aplicación, 1 = con aplicación). Las notaciones 0:12 y 1:12 indican un balance perfecto: 12 parcelas por nivel en cada nutriente.

3.7 Estadísticas descriptivas por factor

Código
factores <- list(
  list(var = "N", nombre = "Nitrógeno"),
  list(var = "P", nombre = "Fósforo"),
  list(var = "K", nombre = "Potasio")
)

tabla_desc <- map_dfr(factores, function(f) {
  npk %>%
    group_by(Nivel = .data[[f$var]]) %>%
    summarise(
      n      = n(),
      Media  = mean(yield),
      Mediana = median(yield),
      DE     = sd(yield),
      Min    = min(yield),
      Max    = max(yield),
      .groups = "drop"
    ) %>%
    mutate(Factor = f$nombre, Nivel = as.character(Nivel)) %>%
    select(Factor, Nivel, everything())
})

4 Especificación del modelo

El modelo se ajusta con el bloque como factor de control y los tres nutrientes con sus interacciones como efectos de tratamiento:

modelo <- aov(yield ~ block + N * P * K, data = npk)

Los resultados del ANOVA indican lo siguiente:

  • Nitrógeno (\(p = 0.0043\)): factor más determinante del rendimiento. Su efecto es altamente significativo.
  • Fósforo (\(p = 0.4749\)): no significativo bajo las condiciones del ensayo. La aplicación de P por sí sola no produjo cambios estadísticamente comprobables en el rendimiento.
  • Potasio (\(p = 0.0288\)): efecto significativo, aunque menos pronunciado que el de N.
  • Interacciones dobles y triple: ninguna resultó estadísticamente significativa (\(p > 0.05\)), lo que indica que los factores actúan de manera independiente en este experimento.

5 Validación de supuestos

5.1 Panel de diagnóstico gráfico

Antes de los tests formales, el análisis gráfico de residuos es la herramienta más informativa para evaluar el cumplimiento de los supuestos del modelo.

Código
autoplot(modelo,
         which  = 1:4,
         ncol   = 2,
         label.size = 3,
         colour = "#1f5e63",
         smooth.colour = "#ffb162",
         smooth.linetype = "dashed") +
  theme_minimal() +
  theme(
    panel.grid.major = element_blank(),
    panel.grid.minor = element_blank(),
    plot.title = element_text(face = "bold", size = 11)
  )

Los cuatro gráficos permiten evaluar:

  • Residuals vs Fitted: la dispersión de los residuos es homogénea a lo largo de los valores ajustados, sin tendencias sistemáticas.
  • Q-Q de residuos: los puntos siguen la línea teórica de normalidad de manera satisfactoria.
  • Scale-Location: la raíz cuadrada de los residuos estandarizados no muestra tendencia creciente ni decreciente (homocedasticidad visual).
  • Cook’s Distance: ninguna observación supera valores de influencia preocupantes.

5.2 Normalidad de residuos — Test de Shapiro-Wilk

shapiro.test(modelo$residuals)

    Shapiro-Wilk normality test

data:  modelo$residuals
W = 0.96875, p-value = 0.6363

La hipótesis nula (\(H_0\)) postula que los residuos siguen una distribución normal. Con \(p\text{-valor} = 0.6363 \gg \alpha = 0.05\), no hay evidencia para rechazar \(H_0\). El supuesto de normalidad se cumple.

5.3 Homocedasticidad — Test de Levene

leveneTest(yield ~ N * P * K, data = npk)
Levene's Test for Homogeneity of Variance (center = median)
      Df F value Pr(>F)
group  7  0.3301 0.9287
      16

La hipótesis nula (\(H_0\)) postula igualdad de varianzas entre los grupos de tratamiento. Con \(p\text{-valor} = 0.9287 \gg \alpha = 0.05\), no se rechaza \(H_0\). El supuesto de homocedasticidad se cumple.

Ambos supuestos del modelo ANOVA quedan validados tanto gráfica como formalmente. Las inferencias realizadas a partir del modelo son estadísticamente válidas.

# Especificación del modelo

El modelo se ajusta con el bloque como factor de control y los tres nutrientes con sus interacciones como efectos de tratamiento:

```{r}
modelo <- aov(yield ~ block + N * P * K, data = npk)
```

## Tabla ANOVA

```{r}
#| code-fold: true
#| code-summary: "Código"

tidy(anova(modelo)) %>%
  kable(
    digits     = 4,
    col.names  = c("Factor", "gl", "SC", "CM", "Estadístico F", "p-valor"),
    caption    = "<span style='color:black; font-weight:bold;
                  font-size:16px;'>Tabla ANOVA — Modelo DBCA Factorial 2³</span>",
    escape     = FALSE,
    align      = c("l", "c", "c", "c", "c", "c")
  ) %>%
  kable_styling(
    bootstrap_options = c("striped", "hover", "condensed"),
    full_width        = FALSE,
    position          = "center",
    font_size         = 13
  ) %>%
  row_spec(0, bold = TRUE, color = "white", background = "#1B2632") %>%
  column_spec(1, bold = TRUE, border_right = TRUE) %>%
  column_spec(6, bold = TRUE) %>%
  row_spec(c(2, 4), background = "#eef4f0")  # N y K significativos
```

Los resultados del ANOVA indican lo siguiente:

-   **Nitrógeno** ($p = 0.0043$): factor más determinante del rendimiento. Su efecto es altamente significativo.
-   **Fósforo** ($p = 0.4749$): no significativo bajo las condiciones del ensayo. La aplicación de P por sí sola no produjo cambios estadísticamente comprobables en el rendimiento.
-   **Potasio** ($p = 0.0288$): efecto significativo, aunque menos pronunciado que el de N.
-   **Interacciones dobles y triple**: ninguna resultó estadísticamente significativa ($p > 0.05$), lo que indica que los factores actúan de manera independiente en este experimento.

# Validación de supuestos

## Panel de diagnóstico gráfico

Antes de los tests formales, el análisis gráfico de residuos es la herramienta más informativa para evaluar el cumplimiento de los supuestos del modelo.

```{r}
#| fig-height: 6.5
#| code-fold: true
#| code-summary: "Código"

autoplot(modelo,
         which  = 1:4,
         ncol   = 2,
         label.size = 3,
         colour = "#1f5e63",
         smooth.colour = "#ffb162",
         smooth.linetype = "dashed") +
  theme_minimal() +
  theme(
    panel.grid.major = element_blank(),
    panel.grid.minor = element_blank(),
    plot.title = element_text(face = "bold", size = 11)
  )
```

Los cuatro gráficos permiten evaluar:

-   **Residuals vs Fitted**: la dispersión de los residuos es homogénea a lo largo de los valores ajustados, sin tendencias sistemáticas.
-   **Q-Q de residuos**: los puntos siguen la línea teórica de normalidad de manera satisfactoria.
-   **Scale-Location**: la raíz cuadrada de los residuos estandarizados no muestra tendencia creciente ni decreciente (homocedasticidad visual).
-   **Cook's Distance**: ninguna observación supera valores de influencia preocupantes.

## Normalidad de residuos — Test de Shapiro-Wilk

```{r}
shapiro.test(modelo$residuals)
```

La hipótesis nula ($H_0$) postula que los residuos siguen una distribución normal. Con $p\text{-valor} = 0.6363 \gg \alpha = 0.05$, no hay evidencia para rechazar $H_0$. **El supuesto de normalidad se cumple.**

## Homocedasticidad — Test de Levene

```{r}
leveneTest(yield ~ N * P * K, data = npk)
```

La hipótesis nula ($H_0$) postula igualdad de varianzas entre los grupos de tratamiento. Con $p\text{-valor} = 0.9287 \gg \alpha = 0.05$, no se rechaza $H_0$. **El supuesto de homocedasticidad se cumple.**

> Ambos supuestos del modelo ANOVA quedan validados tanto gráfica como formalmente. Las inferencias realizadas a partir del modelo son estadísticamente válidas.
